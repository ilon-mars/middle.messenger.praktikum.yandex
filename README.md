## Установка

Сборка проекта. Используемый сборщик [Vite](https://vitejs.dev/):

```bash
yarn build
```

Сборка и запуск проекта:

```bash
yarn start
```

Запуск проекта в режиме разработки:

```bash
yarn dev
```
